> A complete Machine Learning pipeline to analyze banking marketing campaigns and predict customer subscription behavior for term deposits.

---

## ✨ Description

This project performs an **end-to-end Machine Learning workflow** on a banking dataset to predict whether a customer will subscribe to a term deposit.

It includes:

- 📊 Data Analysis & Visualization  
- 🧹 Data Preprocessing & Feature Engineering  
- ⚖️ Handling Imbalanced Data  
- 🤖 Training Multiple ML Models  
- 📈 Performance Evaluation  

---

## 🚀 Features

- 📊 **Exploratory Data Analysis (EDA)** with visual insights  
- 🔄 **Data Cleaning & Encoding** for categorical variables  
- ⚖️ **Class Balancing** using sampling techniques  
- 🤖 **Multiple ML Models** comparison  
- 📈 **Performance Metrics** (Accuracy, Confusion Matrix, Reports)  
- 🔍 **Feature Importance** using Fisher Score  
- 📉 **Correlation Analysis** using heatmaps
