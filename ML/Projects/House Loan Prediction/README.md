> A Machine Learning project to predict whether a loan application will be approved based on applicant details and financial history.

---

## ✨ Description

The **House Loan Prediction System** is an end-to-end ML project that analyzes applicant data to determine loan approval status.

This helps financial institutions:
- Reduce risk 📉  
- Automate decision-making ⚡  
- Improve loan approval accuracy 🎯  

---

- 📊 **Exploratory Data Analysis (EDA)**  
- 🧹 **Data Cleaning & Missing Value Handling**

- 📈 **Model Evaluation & Comparison**

## 📂 Dataset

- Loan Prediction Dataset  
- Features include:
  - Applicant Income  
  - Coapplicant Income  
  - Loan Amount  
  - Loan Term  
  - Credit History  
  - Gender, Education, Property Area  

🎯 Target Variable:
- `Loan_Status` → Approved (Y/N)
